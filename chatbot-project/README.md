# react-chatBot

Interactive chatbot application built with React and Vite.

## Scripts

- `npm run dev` starts the development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build locally.

## Deployment

The project is built as a static site and can be deployed to any static hosting service.

1. Run `npm run build` to create the production build in the `dist` folder.
2. Deploy the contents of the `dist` folder to your hosting service (e.g., Netlify, Vercel, GitHub Pages).

For example, with Vercel:
- Install Vercel CLI: `npm i -g vercel`
- Run `vercel` in the project root and follow the prompts.

For Netlify, simply drag and drop the `dist` folder to the Netlify dashboard.
